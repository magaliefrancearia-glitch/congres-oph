Etape 1

Synchronisation cloud

Pour que toutes les bornes partagent les scans en temps reel, connectez une base Firebase (gratuit). Sinon, chaque appareil fonctionne en autonome.

Comment obtenir l'URL (2 minutes) :

1. Connectez-vous sur console.firebase.google.com
2. Cliquez "Creer un projet" → nommez-le (ex: "congres-oph") → Continuer
3. Dans le menu a gauche : Creer > Realtime Database
4. Cliquez "Creer une base de donnees" → choisissez "mode test"
5. Copiez l'URL qui apparait en haut (elle ressemble a https://congres-oph-default-rtdb.europe-west1.firebasedatabase.app)
✓
Firebase connecte !
